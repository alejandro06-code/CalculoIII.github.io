# Archivos de recursos

Esta carpeta queda reservada para guardar archivos definitivos del curso cuando deban quedar versionados en el repositorio.

Organizacion sugerida:

- `modulo-01/leccion-01/preparacion/`
- `modulo-01/leccion-01/profundizacion/`
- `modulo-01/leccion-01/reto/`
- `modulo-01/leccion-01/cierre/`

La pagina estatica puede adjuntar archivos desde el navegador y descargarlos desde cada recurso, pero GitHub Pages no puede escribir automaticamente nuevos archivos dentro del repositorio. Para materiales oficiales o pesados, guarda el archivo en esta carpeta y registra su enlace o ruta en el recurso.
